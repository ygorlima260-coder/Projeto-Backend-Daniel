Alfabetização no Brasil

Este documento foi construído para apresentar os resultados da triagem e estruturação de dados de geolocalização e administração escolar no Brasil. O fluxo de trabalho consistiu em:
Tratamento: Limpeza e agregação dos dados brutos utilizando a biblioteca Pandas.
Armazenamento: Hospedagem do arquivo final estruturado em um repositório no GitHub.
Visualização: Consumo do arquivo via requisição e renderização dos gráficos interativos aqui no Observable.
Abaixo, detalhamos a estrutura do arquivo de dados utilizado como base para todas as visualizações deste notebook.
O objetivo principal é entender como as escolas se distribuem entre as zonas urbanas e rurais, além de mapear a presença das redes estadual, federal, municipal e privada ao longo das cinco regiões do país.
